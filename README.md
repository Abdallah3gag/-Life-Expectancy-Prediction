# Life Expectancy Prediction

Predict Life Expectancy of countries using WHO dataset and machine learning models.

## Project Overview
- Dataset: WHO Life Expectancy Data
- Rows: 2938, Features: 22
- Target: Life Expectancy

## Features
- Health: BMI, Alcohol, Polio, Diphtheria, HIV/AIDS, etc.
- Demographic: Adult Mortality, Infant deaths, Under-five deaths
- Economic: GDP, Total Expenditure, Income composition
- Education: Schooling

## Machine Learning Models
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- XGBoost Regressor

## Evaluation Metrics
- RMSE, MSE, R² Score

## Results
- XGBoost gave the best performance: RMSE=1.632, R²=0.969
